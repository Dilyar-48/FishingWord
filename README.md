# FishingWord
### Приложение, определяющее положение определённых видов рыб в ближайшей к пользователю точке, показывая путь на карте и примерное расстояние между пунктами. С пользователей собирается статистика: количество пойманных рыб, наживка, клёв в определённую погоду. Так же можно выбрать дату поездки и узнать погоду на соответствующий день. Вероятно добавление: возможности поиска изображения рыбы и добавления его в базу данных, создания коллективной поездки.

#### Техническое задание:
##### 1) Создание формы регистрации и входа
##### 2) Добавить карту, на которой будет отображаться маршрут до водоёма (или отображение дистанции по прямой)
##### 3) Рассчёт растояния между точками
##### 4) Просмотр и заполнение базы рыб
##### 5) Погода в заданном пункте
##### 6) Просмотр данных о поездках других людей в те или иные дни

#### Дополнительные функции, которые можно реализовать:
##### 1) Возможнось создания групповой поездки (сбор команды)
##### 2) запрос изображения рыбы по её названию и наоборот.
