# FishingWord
# Приложение, определяющее положение определённых видов рыб в ближайшей к пользователю точке, показывая путь на карте и примерное расстояние между пунктами. С пользователей собирается статистика: количество пойманных рыб, наживка, клёв в определённую погоду. Так же можно выбрать дату поездки и погоду на соответствующий день. Вероятно добавление: возможности поиска изображения рыбы и добавления его в базу данных, создания коллективной поездки.
